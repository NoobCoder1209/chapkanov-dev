A starter chart that already has the boring stuff right: liveness and readiness
probes that match real workloads, sane HPA limits, a PodDisruptionBudget, a
NetworkPolicy that defaults to deny, ExternalSecrets-friendly secret wiring,
and a JSON schema that catches bad values before `helm template` runs.

CI smoke-tests every change against a local `kind` cluster, so the chart never
ships broken.
