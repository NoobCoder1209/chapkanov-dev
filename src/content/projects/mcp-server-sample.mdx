A starter MCP server you can clone, point at any free public API, and have
Claude Desktop calling it within minutes. Stdio transport, two tools wired up
correctly with Zod-typed inputs, error handling that surfaces useful messages
back to the model, and a config snippet for `claude_desktop_config.json`.

MCP is moving fast — this repo tracks the latest stable SDK shape and
re-pins versions when something breaks.
