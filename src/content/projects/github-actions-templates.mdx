A library of `workflow_call` reusable workflows that other repos can drop in
without copy-pasting fifty lines of YAML each time. Pinned action versions,
typed inputs, secret-passing patterns, and a consistent shape across the set:
build → scan → publish.

Designed to be the boring-but-correct default. If you've ever copy-pasted a
release workflow between three repos, this is the thing.
