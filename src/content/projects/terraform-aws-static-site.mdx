The "I just want to host a static site on AWS the right way" module. S3 origin
locked down via Origin Access Control (no public bucket), CloudFront in front
with sane cache rules, ACM for TLS, Route53 for DNS, all wired so a single
`terraform apply` gets you a working site at your domain.

SPA-friendly out of the box: the CloudFront error responses route 403/404 to
`index.html` so client-side routes don't break.
