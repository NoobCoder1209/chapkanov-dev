Point it at a folder of markdown, run one command, ask a question. The answer
streams back with citations to the source files and chunks.

The whole thing runs locally: Qdrant in Docker for the vector store, a small
embedder to chunk + embed, and Claude on top to answer over the retrieved
context. No SaaS dependency beyond the LLM call itself, which makes it cheap
to iterate on the prompt and the chunking strategy.
